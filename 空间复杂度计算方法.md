# 空间复杂度计算方法

## 核心规则

### 1. 只计算额外空间
```javascript
function sum(arr) {
    let total = 0;  // O(1) 只用了一个变量
    for (let i = 0; i < arr.length; i++) {
        total += arr[i];
    }
    return total;
}
// 空间复杂度：O(1)
// 不计算输入数组 arr 占用的空间
```

### 2. 新建数据结构计入空间
```javascript
function double(arr) {
    const result = [];  // O(n) 新建了长度为 n 的数组
    for (let i = 0; i < arr.length; i++) {
        result.push(arr[i] * 2);
    }
    return result;
}
// 空间复杂度：O(n)
```

### 3. 递归栈空间也要计算
```javascript
function factorial(n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);  // 递归深度 n
}
// 空间复杂度：O(n) - 递归调用栈
```

## 完整计算规则表

| 规则 | 说明 | 示例 |
|------|------|------|
| **1. 变量/常量** | 固定数量的变量 | `let a, b, c` → O(1) |
| **2. 数组/对象** | 新建数据结构按大小计算 | `new Array(n)` → O(n) |
| **3. 递归调用栈** | 递归深度 | 深度为 n → O(n) |
| **4. 嵌套结构** | 多维数组按总元素计算 | `m×n` 矩阵 → O(m×n) |
| **5. 就地修改** | 不新建结构，修改原数据 | 数组反转 → O(1) |
| **6. 取最高阶** | 多个结构取最大的 | O(n) + O(1) → O(n) |

## 实战示例

### 示例1：只用变量 - O(1)
```javascript
function findMax(arr) {
    let max = arr[0];  // O(1)
    let index = 0;     // O(1)
    
    for (let i = 1; i < arr.length; i++) {
        if (arr[i] > max) {
            max = arr[i];
            index = i;
        }
    }
    return max;
}
// 空间复杂度：O(1) ✅
// 只用了 max 和 index 两个变量
```

### 示例2：新建数组 - O(n)
```javascript
function reverseArray(arr) {
    const result = [];  // O(n) 新建数组
    for (let i = arr.length - 1; i >= 0; i--) {
        result.push(arr[i]);
    }
    return result;
}
// 空间复杂度：O(n) ✅
```

### 示例3：就地修改 - O(1)
```javascript
function reverseInPlace(arr) {
    let left = 0, right = arr.length - 1;  // O(1)
    
    while (left < right) {
        // 交换元素，不创建新数组
        [arr[left], arr[right]] = [arr[right], arr[left]];
        left++;
        right--;
    }
    return arr;
}
// 空间复杂度：O(1) ✅
// 直接修改原数组，只用了两个变量
```

### 示例4：递归 - O(n)
```javascript
// 递归计算阶乘
function factorial(n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
// 空间复杂度：O(n) ✅
// 递归栈深度为 n

// 调用过程（n=4）：
// factorial(4)
//   factorial(3)
//     factorial(2)
//       factorial(1)  ← 最深处
// 栈深度 = 4
```

### 示例5：多个数据结构 - 取最大
```javascript
function process(arr) {
    const set = new Set();     // 最坏 O(n)
    const temp = [];           // 最坏 O(n)
    let count = 0;             // O(1)
    
    for (let num of arr) {
        if (!set.has(num)) {
            set.add(num);
            temp.push(num);
        }
        count++;
    }
    return temp;
}
// 空间复杂度：O(n) + O(n) + O(1) = O(2n) = O(n) ✅
```

### 示例6：二维数组 - O(m×n)
```javascript
function createMatrix(m, n) {
    const matrix = [];  // O(m×n)
    
    for (let i = 0; i < m; i++) {
        matrix[i] = [];
        for (let j = 0; j < n; j++) {
            matrix[i][j] = 0;
        }
    }
    return matrix;
}
// 空间复杂度：O(m×n) ✅
// m 行，每行 n 个元素
```

### 示例7：递归 + 额外数组 - O(n)
```javascript
// 快速排序
function quickSort(arr) {
    if (arr.length <= 1) return arr;
    
    const pivot = arr[0];
    const left = [];   // O(n)
    const right = [];  // O(n)
    
    for (let i = 1; i < arr.length; i++) {
        if (arr[i] < pivot) {
            left.push(arr[i]);
        } else {
            right.push(arr[i]);
        }
    }
    
    return [...quickSort(left), pivot, ...quickSort(right)];
}
// 空间复杂度：O(n log n) ⚠️
// 递归深度 O(log n) × 每层 O(n) = O(n log n)
```

## 常见算法空间复杂度分析

### 1. 排序算法

```javascript
// 冒泡排序 - O(1)
function bubbleSort(arr) {
    for (let i = 0; i < arr.length; i++) {
        for (let j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}
// 空间：O(1) - 就地排序

// 归并排序 - O(n)
function mergeSort(arr) {
    if (arr.length <= 1) return arr;
    
    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));
    
    return merge(left, right);  // merge 需要 O(n) 空间
}
// 空间：O(n) - 需要额外数组合并
```

### 2. 查找算法

```javascript
// 二分查找（迭代）- O(1)
function binarySearch(arr, target) {
    let left = 0, right = arr.length - 1;
    
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        if (arr[mid] === target) return mid;
        if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
// 空间：O(1)

// 二分查找（递归）- O(log n)
function binarySearchRecursive(arr, target, left = 0, right = arr.length - 1) {
    if (left > right) return -1;
    
    const mid = Math.floor((left + right) / 2);
    if (arr[mid] === target) return mid;
    if (arr[mid] < target) return binarySearchRecursive(arr, target, mid + 1, right);
    return binarySearchRecursive(arr, target, left, mid - 1);
}
// 空间：O(log n) - 递归栈
```

### 3. 图/树遍历

```javascript
// BFS - O(n)
function bfs(root) {
    const queue = [root];    // O(n) - 队列最多存储一层的节点
    const result = [];       // O(n) - 存储结果
    
    while (queue.length) {
        const node = queue.shift();
        result.push(node.val);
        
        if (node.left) queue.push(node.left);
        if (node.right) queue.push(node.right);
    }
    return result;
}
// 空间：O(n)

// DFS（递归）- O(h)
function dfs(root) {
    if (!root) return [];
    return [root.val, ...dfs(root.left), ...dfs(root.right)];
}
// 空间：O(h) - h 为树高度
// 平衡树：O(log n)
// 最坏情况（链式）：O(n)
```

### 4. 动态规划

```javascript
// 斐波那契（递归 + 记忆化）- O(n)
function fib(n, memo = {}) {
    if (n <= 1) return n;
    if (memo[n]) return memo[n];
    
    memo[n] = fib(n - 1, memo) + fib(n - 2, memo);
    return memo[n];
}
// 空间：O(n) - memo 对象 + 递归栈

// 斐波那契（迭代）- O(1)
function fibIterative(n) {
    if (n <= 1) return n;
    let prev = 0, curr = 1;
    
    for (let i = 2; i <= n; i++) {
        [prev, curr] = [curr, prev + curr];
    }
    return curr;
}
// 空间：O(1) - 只用两个变量

// 背包问题（二维数组）- O(n×m)
function knapsack(weights, values, capacity) {
    const n = weights.length;
    const dp = Array(n + 1).fill(0).map(() => Array(capacity + 1).fill(0));
    
    for (let i = 1; i <= n; i++) {
        for (let w = 0; w <= capacity; w++) {
            if (weights[i - 1] <= w) {
                dp[i][w] = Math.max(
                    dp[i - 1][w],
                    dp[i - 1][w - weights[i - 1]] + values[i - 1]
                );
            } else {
                dp[i][w] = dp[i - 1][w];
            }
        }
    }
    return dp[n][capacity];
}
// 空间：O(n×m)

// 背包问题（空间优化）- O(m)
function knapsackOptimized(weights, values, capacity) {
    const dp = Array(capacity + 1).fill(0);
    
    for (let i = 0; i < weights.length; i++) {
        for (let w = capacity; w >= weights[i]; w--) {
            dp[w] = Math.max(dp[w], dp[w - weights[i]] + values[i]);
        }
    }
    return dp[capacity];
}
// 空间：O(m) - 只用一维数组
```

## 实际题目分析

### 示例1：两数之和

```javascript
// 方法1：暴力 - O(1)
function twoSum1(nums, target) {
    for (let i = 0; i < nums.length; i++) {
        for (let j = i + 1; j < nums.length; j++) {
            if (nums[i] + nums[j] === target) {
                return [i, j];
            }
        }
    }
}
// 空间：O(1) ✅

// 方法2：哈希表 - O(n)
function twoSum2(nums, target) {
    const map = new Map();  // O(n)
    
    for (let i = 0; i < nums.length; i++) {
        const complement = target - nums[i];
        if (map.has(complement)) {
            return [map.get(complement), i];
        }
        map.set(nums[i], i);
    }
}
// 空间：O(n) - 用空间换时间 ✅
```

### 示例2：反转链表

```javascript
// 迭代 - O(1)
function reverseList(head) {
    let prev = null;
    let curr = head;
    
    while (curr) {
        const next = curr.next;
        curr.next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}
// 空间：O(1) ✅

// 递归 - O(n)
function reverseListRecursive(head) {
    if (!head || !head.next) return head;
    
    const newHead = reverseListRecursive(head.next);
    head.next.next = head;
    head.next = null;
    return newHead;
}
// 空间：O(n) - 递归栈 ✅
```

### 示例3：LRU 缓存

```javascript
class LRUCache {
    constructor(capacity) {
        this.capacity = capacity;
        this.cache = new Map();  // O(capacity)
    }
    
    get(key) {
        if (!this.cache.has(key)) return -1;
        const val = this.cache.get(key);
        this.cache.delete(key);
        this.cache.set(key, val);
        return val;
    }
    
    put(key, value) {
        if (this.cache.has(key)) {
            this.cache.delete(key);
        }
        this.cache.set(key, value);
        if (this.cache.size > this.capacity) {
            this.cache.delete(this.cache.keys().next().value);
        }
    }
}
// 空间：O(capacity) ✅
// Map 最多存储 capacity 个元素
```

## 空间优化技巧

### 1. 使用变量代替数组
```javascript
// ❌ 空间 O(n)
function sum(arr) {
    const products = arr.map(x => x * 2);
    return products.reduce((a, b) => a + b, 0);
}

// ✅ 空间 O(1)
function sumOptimized(arr) {
    let total = 0;
    for (let num of arr) {
        total += num * 2;
    }
    return total;
}
```

### 2. 就地修改代替新建
```javascript
// ❌ 空间 O(n)
function rotate(arr, k) {
    return [...arr.slice(-k), ...arr.slice(0, -k)];
}

// ✅ 空间 O(1)
function rotateInPlace(arr, k) {
    k = k % arr.length;
    reverse(arr, 0, arr.length - 1);
    reverse(arr, 0, k - 1);
    reverse(arr, k, arr.length - 1);
    return arr;
}
```

### 3. 迭代代替递归
```javascript
// ❌ 空间 O(n) - 递归栈
function sumRecursive(n) {
    if (n <= 1) return n;
    return n + sumRecursive(n - 1);
}

// ✅ 空间 O(1)
function sumIterative(n) {
    return n * (n + 1) / 2;  // 数学公式
}
```

### 5. 位运算节省空间
```javascript
// ❌ 空间 O(n) - 使用 Set
function findDuplicate(nums) {
    const seen = new Set();
    for (let num of nums) {
        if (seen.has(num)) return num;
        seen.add(num);
    }
}

// ✅ 空间 O(1) - 使用索引标记（修改原数组）
function findDuplicateOptimized(nums) {
    for (let i = 0; i < nums.length; i++) {
        const index = Math.abs(nums[i]);
        if (nums[index] < 0) return index;
        nums[index] = -nums[index];
    }
}
```

## 常见空间复杂度对比

| 复杂度 | 说明 | 示例 |
|--------|------|------|
| O(1) | 常数空间 | 几个变量，就地修改 |
| O(log n) | 对数空间 | 递归二分查找，平衡树递归 |
| O(n) | 线性空间 | 哈希表，额外数组，一维DP |
| O(n log n) | 线性对数 | 递归归并排序（栈+临时数组） |
| O(n²) | 平方空间 | 二维矩阵，图的邻接矩阵 |

**优化顺序（从优到劣）：**
```
O(1) > O(log n) > O(n) > O(n log n) > O(n²)
```

## 总结

**空间复杂度计算要点：**

1. **只计算额外空间**：不包括输入数据本身
2. **新建结构必计**：数组、对象、Set、Map 等
3. **递归栈要算**：递归深度即为栈空间
4. **就地修改最优**：O(1) 空间是最好的
5. **取最高阶项**：多个结构取最大的

**优化策略：**
- ✅ 用变量代替数组
- ✅ 就地修改原数据
- ✅ 迭代代替递归
- ✅ 滚动数组优化DP
- ✅ 权衡时间空间（有时用空间换时间更优）
