# 时间复杂度计算方法

## 核心规则

### 1. 顺序执行（同层级）→ 相加
```javascript
for (let i = 0; i < n; i++) { }  // O(n)
for (let j = 0; j < m; j++) { }  // O(m)

// 总复杂度：O(n) + O(m) = O(n + m)
```

### 2. 嵌套循环（多层级）→ 相乘
```javascript
for (let i = 0; i < n; i++) {      // O(n)
    for (let j = 0; j < m; j++) {  // O(m)
    }
}

// 总复杂度：O(n) × O(m) = O(n × m)
```

### 3. 取最高阶项，忽略常数和低阶项
```javascript
O(n²) + O(n) + O(1) = O(n²)  // 保留最高阶
O(3n² + 5n + 100) = O(n²)    // 去掉常数系数
```

---

## 完整计算规则表

| 规则 | 说明 | 示例 |
|------|------|------|
| **1. 顺序代码相加** | 同级别的代码块时间相加 | `O(n) + O(m)` |
| **2. 嵌套循环相乘** | 循环嵌套时复杂度相乘 | `O(n) × O(m) = O(nm)` |
| **3. 取最高阶** | 只保留增长最快的项 | `O(n² + n) → O(n²)` |
| **4. 忽略常数** | 去掉常数系数和常数项 | `O(3n + 5) → O(n)` |
| **5. 分支取最坏** | if-else 取复杂度最大的分支 | `max(O(n), O(m))` |

## 实战示例

### 示例1：多个顺序循环
```javascript
for (let i = 0; i < n; i++) { }     // O(n)
for (let i = 0; i < n; i++) { }     // O(n)
for (let i = 0; i < n; i++) { }     // O(n)

// 计算：O(n) + O(n) + O(n) = O(3n) = O(n) ✅
```

### 示例2：多层嵌套
```javascript
for (let i = 0; i < n; i++) {           // O(n)
    for (let j = 0; j < n; j++) {       // O(n)
        for (let k = 0; k < n; k++) {   // O(n)
        }
    }
}

// 计算：O(n) × O(n) × O(n) = O(n³) ✅
```

### 示例3：混合情况
```javascript
for (let i = 0; i < n; i++) {       // O(n)
    for (let j = 0; j < n; j++) {   // O(n)
    }
}                                    // = O(n²)

for (let i = 0; i < n; i++) { }     // O(n)

// 计算：O(n²) + O(n) = O(n²) ✅
```

### 示例4：条件分支
```javascript
if (condition) {
    for (let i = 0; i < n; i++) { }         // O(n)
} else {
    for (let i = 0; i < n; i++) {           // O(n)
        for (let j = 0; j < n; j++) { }     // O(n)
    }                                        // = O(n²)
}

// 计算：取最坏情况 O(max(n, n²)) = O(n²) ✅
```

## 实际题目分析：73. 矩阵置零

```javascript
var setZeroes = function (matrix) {
    const m = matrix.length;
    const n = matrix[0].length;
    
    // 步骤1：检查第0行
    for (let j = 0; j < n; j++) { }     // O(n)
    
    // 步骤2：检查第0列
    for (let i = 0; i < m; i++) { }     // O(m)
    
    // 步骤3：标记
    for (let i = 1; i < m; i++) {       // O(m)
        for (let j = 1; j < n; j++) {   // O(n)
        }
    }                                    // = O(m × n)
    
    // 步骤4：置零
    for (let i = 1; i < m; i++) {       // O(m)
        for (let j = 1; j < n; j++) {   // O(n)
        }
    }                                    // = O(m × n)
    
    // 步骤5：处理第0行
    for (let j = 0; j < n; j++) { }     // O(n)
    
    // 步骤6：处理第0列
    for (let i = 0; i < m; i++) { }     // O(m)
};
```

### 计算过程：

```
同层级相加：
O(n) + O(m) + O(m×n) + O(m×n) + O(n) + O(m)
= O(2n + 2m + 2mn)

去掉常数系数：
= O(n + m + mn)

取最高阶项：
= O(mn) ✅
```

---

## 总结

**时间复杂度计算方法：**

1. **同层级**（顺序执行）→ 复杂度**相加**
2. **多层循环**（嵌套）→ 复杂度**相乘**
3. **最终处理**：
   - 去掉常数系数
   - 去掉低阶项
   - 保留最高阶项
4. **条件分支**：取最坏情况

## 常见复杂度对比

| 复杂度 | 名称 | 示例 |
|--------|------|------|
| O(1) | 常数 | 直接访问数组元素 |
| O(log n) | 对数 | 二分查找 |
| O(n) | 线性 | 遍历数组 |
| O(n log n) | 线性对数 | 快速排序、归并排序 |
| O(n²) | 平方 | 双层循环 |
| O(n³) | 立方 | 三层循环 |
| O(2ⁿ) | 指数 | 递归求斐波那契数列（未优化） |
| O(n!) | 阶乘 | 全排列 |

**性能排序（从快到慢）：**
```
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(n³) < O(2ⁿ) < O(n!)
```

## 常见组合操作示例

### 示例1：数组去重
```javascript
// 方法1：使用 Set - 推荐
const unique1 = [...new Set(arr)];
// 时间复杂度：O(n)
// 空间复杂度：O(n)

// 方法2：使用 filter + indexOf
const unique2 = arr.filter((item, index) => arr.indexOf(item) === index);
// 时间复杂度：O(n²)  ❌ 慢
// 原因：filter O(n) × indexOf O(n) = O(n²)
```

### 示例2：数组排序后查找
```javascript
const result = arr.sort((a, b) => a - b).find(x => x > 10);
// 时间复杂度：O(n log n) + O(n) = O(n log n)
// 取最高阶项
```

### 示例3：嵌套数组遍历
```javascript
arr.forEach(item => {
    item.forEach(subItem => {
        console.log(subItem);
    });
});
// 如果每个子数组长度为 m
// 时间复杂度：O(n) × O(m) = O(n × m)
```

### 示例4：链式调用
```javascript
arr.filter(x => x > 0)      // O(n)
   .map(x => x * 2)         // O(n)
   .sort((a, b) => a - b);  // O(n log n)

// 时间复杂度：O(n) + O(n) + O(n log n) = O(n log n)
```

## 性能优化建议

### 1. 选择合适的数据结构
```javascript
// ❌ 使用数组查找（O(n)）
if (arr.includes(value)) { }

// ✅ 使用 Set 查找（O(1)）
const set = new Set(arr);
if (set.has(value)) { }
```

### 2. 避免不必要的排序
```javascript
// ❌ 查找最大值时排序（O(n log n)）
const max = arr.sort((a, b) => b - a)[0];

// ✅ 直接遍历（O(n)）
const max = Math.max(...arr);
```

### 3. 减少嵌套循环
```javascript
// ❌ 双层循环（O(n²)）
for (let i = 0; i < arr1.length; i++) {
    for (let j = 0; j < arr2.length; j++) {
        if (arr1[i] === arr2[j]) { }
    }
}

// ✅ 使用 Map/Set（O(n + m)）
const set = new Set(arr2);
arr1.forEach(item => {
    if (set.has(item)) { }
});
```

### 4. 避免在循环中使用耗时操作
```javascript
// ❌ 每次循环都调用 splice（每次 O(n)，总共 O(n²)）
for (let i = arr.length - 1; i >= 0; i--) {
    if (arr[i] < 0) {
        arr.splice(i, 1);
    }
}

// ✅ 使用 filter（O(n)）
arr = arr.filter(x => x >= 0);
```
